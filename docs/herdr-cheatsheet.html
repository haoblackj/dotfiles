herdr cheat sheet

prefix は ctrl+b、あとは prefix+a だけ覚えればいい

個別キーを全部暗記しなくても、コマンドパレットで名前を打てば大抵のプラグイン操作にたどり着く。 下の検索欄はこのページ自体もパレットと同じ感覚でフィルタできる。

prefix + a

コマンドパレット(fzf)を開く — jt.command-palette

  1. ctrl+b a で起動
  2. action 名の一部を打つ
  3. Enter で実行

ペイン・分割

focus / split / zoom
h左のペインへ
j下のペインへ
k上のペインへ
l右のペインへ
v縦分割
−横分割
zズーム(全画面切替)
xペインを閉じる
bサイドバー切替

タブ

new / switch / rename
c新規タブ
p前のタブ
n次のタブ
1‑9タブ番号で切替
⇧tタブ名変更
⇧xタブを閉じる

ワークスペース

workspace / worktree
wワークスペースピッカー
ggoto(横断移動)
⇧n新規ワークスペース
⇧g新規worktree
⇧wワークスペース名変更
⇧dワークスペースを閉じる

ビュー・メタ

help / settings / detach
?ヘルプ
s設定
qデタッチ
⇧rconfig再読込
o通知ターゲットを開く

プラグイン(専用キーなし・パレット経由)

persiyanov / cloudmanic / jt

個別のキーは割り当てていない。prefix+a でパレットを開き、下の名前の一部を打てば呼び出せる。 直接CLIから叩く場合は herdr plugin action invoke <id>。

persiyanov.reviewr.togglereviewr: 差分レビューサイドバー切替
cloudmanic.herdr-plus.projectsHerdr Plus: プロジェクト起動
cloudmanic.herdr-plus.quick-actionsHerdr Plus: クイックアクション
cloudmanic.herdr-plus.pingHerdr Plus: ping
jt.command-palette.openパレット自体を開く(= prefix+a)

該当なし — prefix+a のパレットでも見つからなければ herdr plugin action list
